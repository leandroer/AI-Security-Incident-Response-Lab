# OWASP GenAI Security Detection Matrix

This matrix maps AI security risks to Microsoft Sentinel detections, required data sources, and response actions.

| Risk / Scenario | Detection Objective | Required Data Sources | KQL Detection | Response |
|---|---|---|---|---|
| LLM01 Prompt Injection | Detect jailbreaks, instruction overrides, hidden prompt extraction attempts, indirect prompt injection payloads | AIApp_CL, AIGateway_CL, AIContentSafety_CL, Azure OpenAI diagnostics, Prompt Shields | `detections/LLM01-Prompt-Injection.kql` | Create Sentinel incident, tag AI-Security/Prompt-Injection, isolate session, disable risky plugins, notify SOC |
| LLM02 Sensitive Information Disclosure | Detect secrets, credentials, tokens, regulated data, or internal prompts in AI responses | AIApp_CL, Microsoft Purview DLP, Defender XDR, OfficeActivity, AIGateway_CL | `detections/LLM02-Sensitive-Information-Disclosure.kql` | Revoke exposed secrets, block session, preserve evidence, notify privacy/security owners |
| LLM06 Excessive Agency | Detect abnormal tool execution, high-risk action chains, and autonomous activity without approval | AIToolExecution_CL, Agent365_CL, Logic App runs, Entra AuditLogs, AzureActivity | `detections/LLM06-Excessive-Agency.kql` | Disable tool/connector, require human approval, suspend agent workflow, assign incident |
| LLM07 System Prompt Leakage | Detect model output containing system/developer instructions or internal policy text | AIApp_CL, AIGateway_CL, AIContentSafety_CL | `detections/LLM07-System-Prompt-Leakage.kql` | Rotate prompts, update guardrails, isolate affected app version, investigate session |
| LLM10 Unbounded Consumption | Detect token abuse, cost spikes, repeated large prompts, or suspicious model invocation volume | AIApp_CL, AzureMetrics, AzureDiagnostics, AIGateway_CL | `detections/LLM10-Unbounded-Consumption.kql` | Rate-limit user/app, disable API key, throttle deployment, notify platform owner |
| Agent Unauthorized Tool Use | Detect agents invoking tools/connectors outside approved policy | Agent365_CL, AIToolExecution_CL, Copilot Studio logs, Logic App runs | `detections/AGENT01-Unauthorized-Tool-Use.kql` | Quarantine or disable agent, remove connector, notify owner |
| Agent Privilege Escalation | Detect agents modifying roles, permissions, policies, or privileged resources | Agent365_CL, Entra AuditLogs, AzureActivity, Defender XDR | `detections/AGENT02-Agent-Privilege-Escalation.kql` | Disable agent, revoke token/session, review permissions, escalate critical |
| Agent Data Exfiltration | Detect agents accessing excessive files, mailboxes, Teams channels, or sensitive data | Agent365_CL, OfficeActivity, Microsoft Purview, Defender for Cloud Apps, SharePoint logs | `detections/AGENT03-Agent-Data-Exfiltration.kql` | Disable agent, suspend connector, preserve logs, revoke delegated permissions |
| Abnormal Agent Execution Volume | Detect sudden agent execution spikes or repeated autonomous tasks | Agent365_CL, AIToolExecution_CL, Logic App runs | `detections/AGENT04-Abnormal-Agent-Execution-Volume.kql` | Rate-limit agent, disable automation, notify SOC and app owner |
| Purview Sensitive Data Exposure | Detect AI or user activity involving labeled or classified data | PurviewDLP_CL, PurviewSensitivity_CL, OfficeActivity, AIApp_CL | `detections/PURVIEW01-Sensitive-Data-Exposure.kql` | Notify SOC and data owner, preserve evidence, restrict data access |
| AI-Assisted Data Exfiltration | Correlate AI/agent activity with mass file access, downloads, external sharing, or DLP matches | Agent365_CL, AIApp_CL, OfficeActivity, PurviewDLP_CL, Defender XDR | `detections/PURVIEW02-AI-Assisted-Data-Exfiltration.kql` | Disable agent/connector, revoke sessions, block external sharing, escalate critical |
| Copilot Oversharing | Detect Copilot/agent interactions involving sensitive documents and broad access permissions | Copilot telemetry, PurviewSensitivity_CL, OfficeActivity, SharePoint logs | `detections/PURVIEW03-Copilot-Oversharing.kql` | Review permissions, apply sensitivity labels, restrict overshared content |
| Agent Access to Regulated Data | Detect Agent 365 or Copilot Studio agents accessing regulated data sources | Agent365_CL, PurviewDLP_CL, PurviewSensitivity_CL, OfficeActivity | `detections/PURVIEW04-Agent-Access-to-Regulated-Data.kql` | Quarantine agent, remove delegated permissions, notify compliance owner |
